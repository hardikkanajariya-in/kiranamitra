## About KiranaMitra

KiranaMitra is a free, offline-first POS & inventory management app built for Indian kirana (grocery) stores. It is designed to be simple, fast, and accessible — no internet or subscription required.

This project is open source. The code is available for learning, contribution, and personal use. **It is not for commercial resale.** See [LICENSE](./LICENSE) for full terms.
